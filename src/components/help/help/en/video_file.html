Let's create a video file!

Once you consider that everything is ready to publish your artwork, it is time to create the video file. This requires you to enter the animation module and give a click on the Export button.

Fig #60. Export option in the Animation module

A dialog box will assist you throughout the export process. The first thing to choose the format you want the video of your animation:

Fig #61. Format Selection Dialog

Then you select the scenes you want to export. In this case, it is only one:

Fig #62. Scene Selection Dialog

Finally, select the file name, the path where you want to export and if you want you can change the dimension in which you want the video to be created:

Fig #63. Dialog File Creation

Once you have clicked the Export button, your video is ready to be seen by everyone! If you want you can post it on our project gallery, just write your wish commenting to go public, following our instructions.

Fig #64. Final video

If you check the source file of this project, you can find here and the video file ready for viewing. This is not a revolutionary work of art, but we hope that motivates you to create great projects!
